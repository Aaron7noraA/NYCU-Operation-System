the input file are in **[Test cases](https://github.com/Aaron7noraA/NYCU-Operation-System/tree/main/LAB2/Test%20cases)**<br>

**A1 ~ A4 is the correct answer of HW2-1 ~ HW2-4**<br>
<br>
<br>
<br>
**compile example**
```c=0
g++ -o HW2-1 FCFS.cpp
```

```c=0
./HW2-1 Test cases/Q1.txt
```
